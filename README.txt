Money Manager Web App (Advanced)
=================================

Files:
- index.html : main HTML file
- style.css  : styling
- script.js  : JavaScript with full functionality

Features:
- Add, delete transactions (amount, description, category, date).
- Stores transactions in browser localStorage (key: money_manager_transactions_v1).
- Advanced filters: by category, by month (YYYY-MM), search text, and sorting.
- Monthly summary: show totals per category for selected month or last 3 months.
- Export transactions to CSV.
- Clear all transactions.

How to use:
1. Open index.html in a browser.
2. Add transactions using the form.
3. Use filters to view specific category/month.
4. Click Export CSV to download all transactions.
5. Transactions are saved locally; they remain after page reload.
